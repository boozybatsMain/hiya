# Фото людей для мокапа телефона

В телефоне-мокапе показываются 4 портрета. Порядок совпадает с рекламным постом
(парень · девушка · парень · девушка):

- 1.jpg — парень (шатен)
- 2.jpg — девушка (блондинка)
- 3.jpg — парень (блонд)
- 4.jpg — девушка (брюнетка)

Форматы: .jpg / .png / .webp (если поменяешь расширение — поправь src в index.html).
Вертикальные портреты 3:4 смотрятся лучше всего (object-fit: cover обрежет как надо).
Исходники ужимаются до ~1100px по большей стороне (sips) — файлы лёгкие (~100–160 КБ).

После добавления/замены:
    git add assets/people
    git commit -m "update photos"
    git push

Пока файлов нет — на их месте аккуратные силуэты-заглушки (страница не выглядит сломанной).
