# Фото людей для мокапа телефона

Положи сюда 4 портрета (те, что ты присылал) с именами:

- 1.jpg  — парень (шатен)
- 2.jpg  — девушка (брюнетка)
- 3.jpg  — парень (блонд)
- 4.jpg  — девушка (блондинка)

Форматы: .jpg / .png / .webp (если поменяешь расширение — поправь src в index.html).
Вертикальные портреты 3:4 смотрятся лучше всего (object-fit: cover обрежет как надо).

После добавления:
    git add assets/people
    git commit -m "add real photos"
    git push

Пока файлов нет — на их месте аккуратные силуэты-заглушки (страница не выглядит сломанной).
