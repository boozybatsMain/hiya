# Пересборка рекламы — очередь задач (TODO)

Задачи, которые выполняются при **следующей пересборке** кампании в Meta Ads.
Копим их здесь, а не правим кампанию по одной, потому что почти любое изменение
адсета (гео, плейсменты, событие оптимизации, бюджет >±20%) перезапускает фазу
обучения — см. «Известные грабли» в [`meta-pixel.md`](meta-pixel.md). Все правки
публикуются одним пакетом.

## Протокол для агента

1. **Перед пересборкой** (или любой пакетной правкой кампании/адсета) открой этот
   файл и **спроси пользователя в чате, какие задачи из списка выполнять**.
   Без явного «да» по задаче — не выполнять.
2. Дальше действуют правила [`CLAUDE.md`](../CLAUDE.md): показать, что именно
   будет сделано, получить подтверждение, опубликовать одним пакетом.
3. Выполненное перенеси в «Журнал» внизу (дата, что сделано, комментарий).
4. Появилась новая идея, требующая правки кампании, — **добавь сюда задачу**
   (что + зачем + дата), а не применяй сразу. Текущий `adset-01-broad` между
   пересборками не трогаем.

## Чертёж пересборки — имена и UTM (заготовка, править тут)

Все имена — ascii-слаги: они через макросы `{{…}}` попадают в UTM и дальше в GA;
кириллица/пробелы дадут кракозябры в отчётах. Имена фиксируются ДО создания
сущностей — переименование после публикации рвёт историю в GA.

| Сущность | Имя | Примечание |
|---|---|---|
| Кампания | `leads-alm-01` | цель «Лиды» |
| Адсет | `adset-01-alm-broad` | Алматы +20–40 км, 20–35, IG-плейсменты вручную, Advantage+ locations выкл, оптимизация на кастомную конверсию OpenSignup |
| Объявление | `creative-01-dark-phone` | текущий креатив (перенос) |
| Объявление | `creative-03-manifest` | Use existing post: пост-манифест |
| Объявление | `creative-04-carousel-how` | Use existing post: карусель «Как это работает» |
| Объявление (резерв) | `creative-05-social-discovery` | фолбэк без слова «знакомства» — если Meta не даст разрешение на dating |

UTM-шаблон (одна строка во ВСЕ объявления, поле «Параметры URL»):

```
utm_source=ig&utm_medium=paid&utm_campaign={{campaign.name}}&utm_content={{ad.name}}&utm_term={{adset.name}}
```

Как это будет выглядеть в GA после подстановки:
`utm_campaign=leads-alm-01`, `utm_content=creative-03-manifest`,
`utm_term=adset-01-alm-broad` — источник/кампания/креатив читаются без декодера.

Чек-лист публикации: старые кампании на паузу тем же пакетом; в новом адсете
снять Advantage+ locations и «Ограниченные затраты на исключённые места
размещения»; проверить, что событие оптимизации — кастомная конверсия
OpenSignup; Boost из приложения не использовать.
