# Пересборка рекламы — очередь задач (TODO)

Задачи, которые выполняются при **следующей пересборке** кампании в Meta Ads.
Копим их здесь, а не правим кампанию по одной, потому что почти любое изменение
адсета (гео, плейсменты, событие оптимизации, бюджет >±20%) перезапускает фазу
обучения — см. «Известные грабли» в [`meta-pixel.md`](meta-pixel.md). Все правки
публикуются одним пакетом.

## Протокол для агента

1. **Перед пересборкой** (или любой пакетной правкой кампании/адсета) открой этот
   файл и **спроси пользователя в чате, какие задачи из списка выполнять**.
   Без явного «да» по задаче — не выполнять.
2. Дальше действуют правила [`CLAUDE.md`](../CLAUDE.md): показать, что именно
   будет сделано, получить подтверждение, опубликовать одним пакетом.
3. Выполненное перенеси в «Журнал» внизу (дата, что сделано, комментарий).
4. Появилась новая идея, требующая правки кампании, — **добавь сюда задачу**
   (что + зачем + дата), а не применяй сразу. Текущий `adset-01-broad` между
   пересборками не трогаем.
